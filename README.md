# NutriAI Demo — Beginner Phone‑First Prototype

NutriAI Demo is a simple learning prototype that accepts health inputs (Glucose, Cholesterol, Iron),
runs a tiny rule engine, and returns a sample meal suggestion with reasoning.
This is a student project and **not medical advice**.

## How to run (on Replit)
1. Open the Replit link for this repo.
2. Click Run.
3. Fill the form with sample values.
4. Click Analyze to see flags and a sample meal day.

## Privacy & disclaimer
- Uploaded PDFs are processed temporarily and deleted after analysis.
- This demo is educational only and not a substitute for professional medical advice.
